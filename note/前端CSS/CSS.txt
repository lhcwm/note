一,CSS层叠样式表:Cascading Style Sheet
  1,功能:HTML负责书写网页结构和内容,CSS调整页面布局和外观
  2,CSS使用方式:
    ①行内样式(内联样式)
      借助style标签属性书写样式代码
      例:<h1 style="color:red;font-size:20px;"></h1>
    ②内嵌样式表
      实现结构与样式的分离
      借助于<style type="text/css"></style>书写CSS样式声明
      例:
        <style>
            选择器{
                属性名:值;
                属性名:值;
                ...
            }
        </style>
      选择器:根据标签名,属性值,层级结构等方式匹配文档中的元素,为其应用样式
      以标签选择器为例:根据标签名匹配文档中所有的该元素
        p{
            color:red;
        }
    ③外部的样式表
      创建.css文件,书写css选择器
      在html中借助于link标签引入外部的样式表
        <link rel="stylesheet" href="" type="text/css">
    
  3,样式表特征:
    ①层叠性:同一个元素应用多种样式共同起作用
    ②继承性:子元素可以继承父元素中的某些样式,例:大部分的文本属性都可以被继承
           超链接的文本色不能由继承得到,需要单独修改
           注意:块元素默认宽度与父元素保持一致
