阶段:web前端
内容:HTML  CSS  JavaScript
--------------------------
一,web开发
  1,web即为网页,由浏览器运行的程序.由浏览器,服务器和协议组成.
    浏览器:代替用户发请求,解析并呈现数据
    服务器:接受请求并响应,存储数据
    协议:规范数据的传输方式
  2,主流浏览器
    根据浏览器的引擎(内核)划分为五大浏览器
    chrom  safari  firefox  IE/Edge  Opera
    浏览器引擎:渲染引擎  JS引擎
二,HTML介绍
  1,超文本标记语言
  2,标记:也叫标签或元素,以< >为标志,用于书写网页结构
    分类:
        双标签:成对出现,有开始有结束
        单标签:只有开始标签
  3,语法规范
    ①标签名不区分大小写,推荐用小写字母
    ②双标签必须写完整,如果缺少了结束标签或开始标签,浏览器会自动补全,可能影响整体布局
    ③所有不使用标签表示的内容,都会自动添加到body中,原样显示到窗口
    ④合理的注释和缩进增加代码的可读性 
      注释语法: <!--注释内容-->
三,常用标签介绍
  1,head中常用标签
    <meta charset='utf-8'> 通过标签属性,设置字符编码
    <title> </title>  网页标题
    <link rel='shortcut icon' href='图片路径' type='image/x-icon'>  设置选项卡图标
  2,body中常用标签
    1,文本标签
      ①标题标签: h1-h6 自带文本加粗效果,字号逐级减小
        <h1>1</h1>
        <h2>2</h2>
        <h3>3</h3>
        <h4>4</h4>
        <h5>5</h5>
        <h6>6</h6>

      ②段落
        <p>自成一段,浏览器默认字体大小为16px</p>

      ③普通文本标签
        <span>处理特殊文本</span>
        <label>有特殊作用</label>

      ④带有格式的标签
        <b>加粗</b>
        <strong>strong表示加粗(有语义标签)</strong>

      ⑤换行标签
        <br/>
    
      ⑤了解内容
        <i>italic表示斜体</i>
        <s>删除线</s>
        <u>underline下划线</u>
        <sub>下角标</sub>
        <sup>上角标</sup>
        例:
        <span>
            x<sub>1</sub>=10
            y<sup>2</sup>
        </span>

        水平线标签
        <hr>

        <h1>浏览器会忽略代码中多余的换行和空格,        只解析为一个空格</h1>

        align标签属性设置文本内容的水平对齐方式,默认局左,可取left/center/right
        <h1 align=''>1</h1>

    2,字符实体:用来处理特殊符号
            &lt;        小于号
            &gt;        大于号
            &nbsp;      空格
            &yen;  ￥   人民币符号
            &copy;  ©   版权符号

            例:
            <h1>&lt;day01&gt;主&nbsp;&nbsp;&nbsp;要讲解标签
            人民币符号&yen;￥
            版权符号：&copy;©

    3,容器标签:常用于页面结构的划分
        <div></div>
    
    4,结构标签:
        有序列表:有序列表(ordered list,默认以阿拉伯数字对每条数据进行编号)
            <ol>
                <li>list item</li>
                <li>列表项</li>
                <li>列表项</li>
                <li>列表项</li>
            </ol>
        无序列表:无序列表(unordered list)
            <ul>
                    <li>列表</li>
                    <li>列表</li>
                    <li>列表</li>
            </ul>
            
        列表属性:
            ol标签:属性type用来设置项目符号类型,取值1,a,A,i,I
                  属性start可以设置从第几个符号开始排列,取无单位的数值
            ul标签 属性 type属性可取disc(默认),circle(空心圆),square(实心方块),none(取消项目符号)
            
            例:<ol type='I'> </ol>

        列表的嵌套:
            列表嵌套推荐使用父子嵌套.能够实现下拉菜单的效果
            例: <ol>
                <li><h2>《三国演义》</h2>
                <ul type='square'>
                <li>刘备</li>
                <li>关羽</li>
                <li>张飞</li>
                </ul>
                </li>
                </ol>

    5,图片与超链接
      1,URL (统一资源定位符--路径)
        组成:协议 域名 文件目录 文件名
        分类:
            绝对路径:从根目录开始逐级查找文件,以盘符或者/开头的路径
            相对路径:从当前所在的文件夹开始查找,使用../表示返回上一级目录

      2,图片标签
        <img src=''>在网页中插入图片
        属性:
            src 必填,设置图片的URL  
            注:①Windows系统下直接复制绝对路径需要手动修改\为/
               ②复制路径只到文件夹,手动补全文件名,必须写后缀

            width/height  设置图片尺寸,取像素值,如果某个方向上缺省,浏览器会根据图片宽高比自适应

            title 设置图片的标题  鼠标悬停在图片上显示
            alt  设置图片加载失败时的提示文本
      
      3,超链接标签
        超链接: 借助文本或图片连接到其他资源文件  ,自带下划线,蓝色是访问前,紫色是访问后
        格式:<a href=''>超链接内容(文本/图片)</a>
        属性:
            href  必填项,指定链接地址   如果是网络资源必须写协议
                  特殊取值:''  空  连接至本页,相当于刷新
                          '#'    连接至本页,不刷新,会改变url,同时#是锚点的标志
                          'javascript:void(0)'  连接至本页,既不会刷新,也不会修改url
            target  设置目标文件的打开方式,默认值_self,表示在当前窗口打开目标文件
                    可取_blank,表示新建窗口打开
            
  3,标签之间的关系
    标签嵌套时,产生祖先与后代的关系,父子关系,兄弟关系
    父子关系:外层元素称为父元素,内层元素成称子元素
    祖先和后代:在多级嵌套结构下,外层所有元素统称为祖先元素,内层元素称为后代元素(直接子元素,间接子元素)
    兄弟关系:平级元素之间为兄弟元素
